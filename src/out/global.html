FAQ(visibleAnswers, setVisibleAnswers, handleClick)

Parameters:
Name Type Description
visibleAnswers Array.<boolean> an array of boolean values used for the logic of hiding or revealing an answer. All answers are hidden by default.
setVisibleAnswers function This function component will be responsible for rendering the list of questions, as well as handling the logic of whether an answer should be revealed (accomplished by using the state hook)
handleClick function a callback function called when one of the buttons is clicked. It is used for revealing the corresponding answer.
